## RUNNING THE TESTS


Navigate to your project file , e.g if the name of your project is my_webscrapping_project
and if it is located in the c: drive root directory

the you will navigate to 

c:\ my_webscrapping_project>



Testing the version of Python 3.x

Make sure you can see the following when you run python at the command prompt
For example 
c:\ my_webscrapping_project>python - version

You should  get something simillar to this

Python 3.7.3rc1 (tags/v3.7.3rc1:69785b2127, Mar 12 2019, 22:37:55) [MSC v.1916 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.

This shows Python is installed

Also run the following Command in cmd to see installed modules
From the   prompt 

For example

c:\ my_webscrapping_project>pip freeze
